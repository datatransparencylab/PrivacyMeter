Thank you for using PrivacyMeter! Privacy online matters, and now can be metered

PrivacyMeter provides a real-time privacy quantification for the modern Web. It is a browser extension that, on-the-fly, computes a relative privacy score for any website that a user is visiting. This score is computed based on each website's privacy practices and how these compare to the privacy practices of many other pre-analyzed websites. Currently, it supports four main widgets, analyzing the trackers present on the page, fingerprinting activity, amount of third-party content, and leaky web forms. In addition, the extension provides users with contextual information about all the discovered privacy issues.

PrivacyMeter is a non-commercial research tool developed by PragSec Lab at Stony Brook University, and proudly powered by EasyList (https://easylist.to), Adblock Plus (https://adblockplus.org), Web of Trust "MyWOT" (https://www.mywot.com), IP2Location LITE (http://lite.ip2location.com), Alexa Top Sites (https://www.alexa.com/topsites), and Highcharts (https://www.highcharts.com).

No personally identifiable information will be collected, and no visited URLs will be shared with third-parties! If "crowdsourcing is enabled", the extension reports to its backend the following privacy practices per web page: list of present trackers, fingerprinting APIs called, list of third-party iframes, and the list of leaky web forms.

Important: even without "crowdsourcing" enabled, extension caches information about trackers on its backend (of course, without any information about actually visited websites).
